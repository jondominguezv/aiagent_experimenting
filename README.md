# AI Agent Experimenting
This repo is for experimenting with generative AI APIs. Currently using Google generative AI.

## Running Agent
`uv run main.py <Prompt>`

## Extras
`calculator/` is a small sample app for the purpose of testing the Agent's code editing by way of the available functions in `functions/`

### Current avaiable functions for the agent
* get_file_content - Reading content from a file.
* get_files_info - Get meta data for files in a directory.
* run_python_file - Run a Python script.
* write_file.py - Write to a file.
